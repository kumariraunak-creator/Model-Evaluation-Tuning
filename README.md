# Titanic Survival Prediction - Model Evaluation & Tuning

## Project Overview

This project evaluates machine learning models on the Titanic dataset and improves performance using hyperparameter tuning.

## Dataset

Preprocessed Titanic Dataset

## Models Used

- Logistic Regression
- Decision Tree Classifier

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score

## Hyperparameter Tuning

GridSearchCV was used to optimize the Decision Tree model.
